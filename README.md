## :checkered_flag: How to run ##

```bash
# Clone this project
$ git clone https://github.com/grBurgos68/portafolio

# Access
$ cd portafolio

# Install dependencies
$ yarn

# Run the project
$ yarn start

# The server will initialize in the <http://localhost:3000>
```
